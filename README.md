# Matematické kyvadlo

Program počíta dĺžku periody matematického kyvadla v závislosti na dĺžke ramena $L$ a gravitačnom zrýchlení $g$

## Matematika


$T = 2 \pi \sqrt(L/g)$

* **T** = perioda (s)
* **L** = dĺžka (m)
* **g** = gravitačné zrýchlenie (m/s²)
